## Overview

This endpoint allows users to export all their data from the Context Zero AI platform in a standardized format. This includes memories, events, permissions, settings, and metadata. The export is useful for data portability, backup purposes, or compliance requirements.

## Query Parameters

<ParamField query="format" type="string" default="json">
  Export format: "json", "csv", "xml"
</ParamField>

<ParamField query="include_archived" type="boolean" default="false">
  Whether to include archived memories in the export
</ParamField>

<ParamField query="include_events" type="boolean" default="true">
  Whether to include event history in the export
</ParamField>

<ParamField query="include_permissions" type="boolean" default="true">
  Whether to include permission settings in the export
</ParamField>

<ParamField query="date_range" type="string">
  Optional date range filter (format: "YYYY-MM-DD,YYYY-MM-DD")
</ParamField>

<ParamField query="compression" type="string" default="gzip">
  Compression format: "none", "gzip", "zip"
</ParamField>

## Response

The response varies based on the requested format:

### JSON Format Response

<ResponseField name="export_metadata" type="object">

<ResponseField name="events" type="array">

<RequestExample>

```bash cURL
curl -X GET "https://api.contextzero.ai/api/v1/util/export?format=json&include_archived=true&compression=gzip" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Accept: application/json" \
  --output user_data_export.json.gz
```

```javascript JavaScript
const response = await fetch('https://api.contextzero.ai/api/v1/util/export?format=json&include_archived=true', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer YOUR_API_TOKEN',
    'Accept': 'application/json',
  },
});

// For JSON format
const exportData = await response.json();

// For file download
const blob = await response.blob();
const url = window.URL.createObjectURL(blob);
const a = document.createElement('a');
a.href = url;
a.download = 'user_data_export.json';
a.click();
```

```python Python
import requests
import json
from datetime import datetime

response = requests.get(
    "https://api.contextzero.ai/api/v1/util/export",
    headers={
        "Authorization": "Bearer YOUR_API_TOKEN",
        "Accept": "application/json"
    },
    params={
        'format': 'json',
        'include_archived': True,
        'include_events': True,
        'compression': 'gzip'
    }
)

# Save to file
filename = f"user_data_export_{datetime.now().strftime('%Y%m%d')}.json"
with open(filename, 'wb') as f:
    f.write(response.content)

print(f"Export saved to {filename}")
```

## Export Formats

<Tabs>
  <Tab title="JSON">
    Structured JSON format with full data fidelity. Best for programmatic processing and re-importing.
  </Tab>
  <Tab title="CSV">
    Comma-separated values format. Best for spreadsheet analysis. Complex nested data is flattened.
  </Tab>
  <Tab title="XML">
    XML format with schema validation. Good for enterprise systems requiring structured markup.
  </Tab>
</Tabs>

## Large Export Handling

For users with large datasets (>100MB), the export process works differently:

<Steps>
  <Step title="Request Initiated">
    Export request is queued for background processing
  </Step>
  <Step title="Processing Notification">
    User receives email notification when processing begins
  </Step>
  <Step title="Download Ready">
    Secure download link is provided when export is complete
  </Step>
  <Step title="Automatic Cleanup">
    Export files are automatically deleted after 7 days
  </Step>
</Steps>
