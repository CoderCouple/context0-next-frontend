## Overview

This endpoint allows you to create permission rules that control access to memories and other resources. Permissions can be granted to users, groups, or applications.

## Request Body

<ParamField body="resource_type" type="string" required>
  Type of resource the permission applies to ("memory", "memory_space", "integration")
</ParamField>

<ParamField body="resource_id" type="string" required>
  ID of the specific resource
</ParamField>

<ParamField body="grantee_type" type="string" required>
  Type of entity receiving the permission ("user", "group", "application", "public")
</ParamField>

<ParamField body="grantee_id" type="string">
  ID of the grantee (required unless grantee_type is "public")
</ParamField>

<ParamField body="permissions" type="array" required>
  Array of permission levels to grant
  <Expandable title="Available Permissions">
    - `read`: View the resource
    - `write`: Modify the resource
    - `delete`: Remove the resource
    - `share`: Grant permissions to others
    - `admin`: Full administrative access
  </Expandable>
</ParamField>

<ParamField body="conditions" type="object">
  Optional conditions for the permission
  <Expandable title="Condition Object">
    <ParamField body="expires_at" type="string">
      ISO timestamp when the permission expires
    </ParamField>
    <ParamField body="ip_restrictions" type="array">
      Array of allowed IP addresses/ranges
    </ParamField>
    <ParamField body="time_restrictions" type="object">
      Time-based access restrictions
    </ParamField>
  </Expandable>
</ParamField>

## Response

<ResponseField name="permission_id" type="string">
  Unique identifier for the created permission
</ResponseField>

<ResponseField name="status" type="string">
  Status of the permission creation
</ResponseField>

<ResponseField name="effective_immediately" type="boolean">
  Whether the permission is immediately effective
</ResponseField>

```python Python
import requests
from datetime import datetime, timedelta

# Create a permission that expires in 6 months
expires_at = (datetime.now() + timedelta(days=180)).isoformat()

response = requests.post(
    "https://api.contextzero.ai/api/v1/permission/",
    headers={
        "Authorization": "Bearer YOUR_API_TOKEN",
        "Content-Type": "application/json"
    },
    json={
        "resource_type": "memory",
        "resource_id": "mem_67890",
        "grantee_type": "user",
        "grantee_id": "user_54321",
        "permissions": ["read", "write"],
        "conditions": {
            "expires_at": expires_at
        }
    }
)

result = response.json()
```

## Permission Levels

<AccordionGroup>
  <Accordion title="Read">
    Allows viewing the resource content and metadata. Does not permit modifications.
  </Accordion>

  <Accordion title="Write">
    Allows modifying the resource content and metadata. Includes read permissions.
  </Accordion>

  <Accordion title="Delete">
    Allows removing the resource permanently. Use with caution.
  </Accordion>

  <Accordion title="Share">
    Allows granting permissions to other users or groups. Enables delegation of access control.
  </Accordion>

  <Accordion title="Admin">
    Full administrative access including all other permissions plus system-level operations.
  </Accordion>
</AccordionGroup>
